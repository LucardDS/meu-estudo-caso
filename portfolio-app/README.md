# Portfolio App

## Overview
This project is a portfolio website designed to showcase specialties, projects, and a contact form. It is structured to provide a clean and responsive user experience, utilizing modern web technologies.

## Project Structure
```
portfolio-app
├── src
│   ├── index.html        # HTML structure of the portfolio page
│   ├── style.css         # CSS styles for the portfolio page
│   ├── menu.js           # JavaScript for mobile menu functionality
│   └── images            # Folder for image assets
├── README.md             # Documentation for the project
```

## Files Description

- **src/index.html**: Contains the HTML structure of the portfolio page, including sections for the header, main content, and footer. It links to the CSS and JavaScript files.

- **src/style.css**: Contains the CSS styles for the portfolio page, defining the layout, colors, typography, and responsive design. The styles reflect the color scheme from the "portfolio" project.

- **src/menu.js**: Contains JavaScript code that manages the mobile menu functionality, adding event listeners to buttons for opening and closing the mobile menu.

- **src/images**: This folder is intended to hold image assets used in the portfolio, such as logos and project images. Place your image files here.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the `portfolio-app` directory.
3. Open the `src/index.html` file in a web browser to view the portfolio.
4. Customize the content and styles as needed to fit your personal branding.
